partforge

Write one declarative part definition — geometry build functions plus a parameter schema — and partforge renders a full parametric-CAD web app: a 3-D viewer, a control panel, geometry workers, and STL / STEP / 3MF export. The pages below are live example parts. Drag to orbit, tweak the controls, download a printable file.
